- [x] Follow User
- [x] Unfollow User
- [ ] Edit my profile
- [ ] See user profile
- [ ] See MY profile
- [ ] See the full post
- [ ] Upload a photo
- [ ] Edit / Delete the photo
- [ ] See the feed
- [ ] Send private Message
- [ ] See rooms
